# Sumedh Patil — Portfolio

A full-stack portfolio website with a contact form backed by MongoDB.

🌐 **Live site:** https://sumedh-portfolio.onrender.com

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js + Express
- Database: MongoDB Atlas
- Deployed on: Render

## Features
- Responsive portfolio with about, projects, skills, and contact sections
- Contact form that saves messages to MongoDB
- Admin panel to view messages (`/admin.html`)
